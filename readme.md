This is a basic C program for windows to create blank files from the command line. To use this, copy `blank.exe` to the folder you want to install it to and make sure that folder is on your path. Any folder should work, within reason.

This link discusses how to edit your path in Windows 7, 8, and 10: https://www.howtogeek.com/118594/how-to-edit-your-system-path-for-easy-command-line-access/

Note: You may encounter issues with your antivirus because this exe is not signed and really looks suspicious. That's because it just opens a file and closes it, making it look like it's trying to hide some bad code.
